# Block Palette

Use namespaced Minecraft ids such as `minecraft:oak_planks`.

## Common building blocks

- Foundation: `minecraft:cobblestone`, `minecraft:stone_bricks`
- Walls: `minecraft:oak_planks`, `minecraft:spruce_planks`, `minecraft:stripped_oak_log`
- Roofs: `minecraft:oak_stairs`, `minecraft:spruce_stairs`, `minecraft:dark_oak_stairs`
- Details: `minecraft:glass_pane`, `minecraft:oak_door`, `minecraft:lantern`
