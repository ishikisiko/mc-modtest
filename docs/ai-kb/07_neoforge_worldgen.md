# NeoForge Worldgen

Worldgen data should live under `src/main/resources/data/<modid>/worldgen/` in a finished mod project.

## Typical folders

- `template_pool/`
- `structure/`
- `structure_set/`

Examples are kept in `docs/ai-kb/examples/worldgen/`.
