# Plaque Frame Brief

This is the v1 frame catalog for custom MyVillage plaques. Frame textures are 16x16 per block part; inscriptions are separate HD painting textures.

| Preset | Register | Size | Vertical | Visual brief |
| --- | --- | --- | --- | --- |
| `town_shop_wood_3w` | town shop | 3w x 1h | 3h | Weathered oak, plain bevel rail, small peg caps, iron eyelets. Rustic and readable. |
| `town_inn_lacquered_4w` | town inn | 4w x 1h | 4h | Red-brown lacquer, rounded rail, brass pin clusters, dark brass eyelets. Warm and hospitable. |
| `town_notice_board_3w` | town civic | 3w x 1h | none | Rough pine board, flat plank edge, black nail plates, iron strap suggestion. Horizontal only. |
| `tavern_signboard_4w` | civic tavern | 4w x 1h | 4h | Dark oak, thick inset rail, diagonal brass corner brackets, paired chain rings. |
| `sect_simple_pine_4w` | sect modest | 4w x 1h | 4h | Pale pine, thin cloud rail, inked cloud corners, wrapped cord loops. |
| `sect_scripture_ornate_4w` | sect scripture | 4w x 1h | 4h | Dark lacquer, jade-green stepped rail, jade cloud plates, green-bronze rings. |
| `lord_manor_heraldry_5w` | civic manor | 5w x 1h | none | Polished walnut, double rail, shield-like enamel corner plates, heavy brass lugs. Horizontal only. |
| `sect_treasure_gilded_5w_2h` | grand sect | 5w x 2h | 5h | Black lacquer, gilded raised rim, cloud-and-pearl bosses, three ring mounts for 5w spans. |

Horizontal parts use `single_left`, `single_center`, `single_right`, `top_left`, `top_center`, `top_right`, `bottom_left`, `bottom_center`, and `bottom_right` as needed by size. Vertical parts use `top`, `middle`, and `bottom`.

Hanging mount textures must include visible ring or eyelet detail on top-left and top-right parts; 5w spans also include top-center hardware.
