# 修行入门录

当前可玩的修行顺序是：先在[入门仪式](./getting_started/initiation.md)中完成测灵，觉醒灵根；再传承《基础吐纳诀》；随后按[打坐与冲关](./getting_started/cultivation_loop.md)的规则先积累修为，再巩固稳定度，最后完成确定性的逐层晋升。

<SubPages icons={true} />

## 当前版本边界

- 测灵与传功是两个独立步骤，学习功法不会自动增加修为或稳定度。
- 普通打坐与灵石修炼都由服务端结算；修为未满时先增加修为，修为圆满后的下一批次才开始增加稳定度。
- 修为和稳定度都达到当前阶段上限后，才可进行一次确定性的晋层或冲关。
- **炼气四层是当前版本的可玩上限。** 炼气五层及之后的修为增长、筑基玩法、丹药、修炼设施、天劫、寿尽死亡与转世尚未实现。

先阅读[入门仪式](./getting_started/initiation.md)，再阅读[打坐与冲关](./getting_started/cultivation_loop.md)。
